CSS Flexbox

Flexbox summary

The flexbox attribute is a great tool for when you're trying to make a responsive website. This is because the flexbox can be used to automatically, space, size, and even wrap elements into multiple columns! One great example of the flexbox is on this website homepage, where the topics being displayed are held inside of a flexbox.
